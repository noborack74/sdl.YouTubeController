# YouTubeController
===

# Overview

YouTubeをspeechRecognizerを使って音声で操作するアプリです。

# Description

"上"/"下"で動画をブラウズ、"再生"で動画を再生、"検索"で動画を検索、"ホーム"/"急上昇"でホーム画面と急上昇画面を行き来できます。

"再生"コマンドは、ピンク色にハイライトされている動画を再生します。


# Requirement

Android上で動きます。

# Usage

メニューバー右の"一時停止中"/"聞いています"をタップすることで、音声認識のオン/オフが切り替えられます。
